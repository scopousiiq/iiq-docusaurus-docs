Retrieves models linked to a specific application integration.

**Prerequisites**
- **AppId** - Application identifier string (not a GUID)

**Use Cases**
- View models associated with an app integration
- App-specific model catalog
- Integration-based model filtering

**Related Endpoints**
- [POST /api/v1.0/models/apps/\{AppId\}](/docs/api/models/get-models-for-app) - Same with POST filtering
- [POST /api/v1.0/models/\{ModelId\}/link/app/\{AppId\}](/docs/api/models/link-model-to-app) - Link model to app
