Updates an existing purchase order record. If the purchase order does not exist, it will be created.

**Updatable Fields**:
- `PoNumber` - PO number/reference
- `IsApproved` - Approval status
- `IsClosed` - Closure status
- `Budget` - Total budget amount
- `Notes` - Additional notes
- `SupplierId` - Associated supplier
- `FundingSourceId` - Funding source
- `PurchaseOrderStatusTypeId` - Status type
- `PurchaseOrderTypeId` - Order type

**Note**: This endpoint is part of the legacy Parts module API which is deprecated.
