Creates a new purchase order record for tracking parts procurement and budget allocation.

**Prerequisites**
1. **SupplierId** (optional) - Use [GET /api/v1.0/parts/suppliers](/docs/api/parts/list-part-suppliers) to list suppliers, extract `Items[].PartSupplierId`.
2. **FundingSourceId** (optional) - Use funding sources API to get available funding sources.

**Workflow Example**
1. (Optional) List suppliers: [GET /api/v1.0/parts/suppliers](/docs/api/parts/list-part-suppliers).
2. Create PO: [POST /api/v1.0/purchaseorders/new](/docs/api/parts/create-purchase-order) with PoNumber, Budget.
3. Verify creation: [GET /api/v1.0/purchaseorders/\{PurchaseOrderId\}](/docs/api/parts/get-purchase-order-by-id).

**Minimal Required Fields**: PoNumber.

**Note**: This endpoint is part of the legacy Parts module API which is deprecated.
